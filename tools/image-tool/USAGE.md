# 🖼️ 图片压缩 & 格式转换工具

> 纯前端图片处理工具，支持压缩、格式转换、尺寸调整、批量处理

## 功能特性

- **图片压缩**：调整质量参数（20%-100%），减小文件体积
- **格式转换**：PNG ↔ JPEG ↔ WebP ↔ BMP 互转
- **尺寸调整**：限制最大宽高，保持原始宽高比
- **批量处理**：同时处理多张图片
- **拖拽上传**：支持拖拽文件到上传区域
- **对比预览**：压缩前后大小对比
- **下载结果**：单张下载或全部下载
- **历史记录**：自动保存处理记录

## 使用方法

### 1. 上传图片

- 点击"选择图片"按钮或拖拽图片到上传区域
- 支持 PNG、JPEG、WebP、BMP、GIF 格式
- 单文件最大 20MB
- 可同时选择多张图片

### 2. 设置处理选项

| 选项 | 说明 |
|------|------|
| 质量 | 图片质量（20%-100%），越高画质越好，文件越大 |
| 输出格式 | 选择输出格式，选"保持原格式"则不转换 |
| 最大宽度 | 限制图片最大宽度，选"不限制"则保持原尺寸 |
| 最大高度 | 限制图片最大高度，选"不限制"则保持原尺寸 |

### 3. 处理图片

点击"全部处理"按钮，等待处理完成。

### 4. 查看结果

- 查看每张图片的压缩前后对比
- 查看压缩率、输出格式、尺寸变化
- 点击"下载"保存单张图片
- 点击"全部下载"保存所有图片

## 技术说明

- 使用 Canvas API 进行图片处理
- 使用 `canvas.toDataURL()` 导出图片
- 所有处理在浏览器本地完成，不上传服务器
- 文件大小估算基于 Base64 编码数据长度

## 浏览器兼容

- Chrome 50+
- Firefox 50+
- Safari 10+
- Edge 15+
- IE 不支持（Canvas toBlob 限制）
