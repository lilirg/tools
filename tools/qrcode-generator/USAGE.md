# 📱 二维码生成器

纯前端 QR 码生成工具，不依赖任何外部库，支持丰富的自定义选项。

## 功能特性

- ✅ **纯前端实现** - 无需后端服务，完全在浏览器中生成
- ✅ **多种编码模式** - 支持数字、字母、字节编码
- ✅ **4 种纠错等级** - L(7%) / M(15%) / Q(25%) / H(30%)
- ✅ **自定义颜色** - 前景色、背景色自由搭配
- ✅ **预设配色** - 12 种精心设计的配色方案
- ✅ **渐变色支持** - 线性渐变前景色
- ✅ **中心 Logo** - 支持上传图片作为二维码中心 Logo
- ✅ **多种点样式** - 方形、圆角、菱形
- ✅ **多种尺寸** - 160px ~ 560px 可调
- ✅ **下载 PNG** - 导出为 PNG 图片
- ✅ **下载 SVG** - 导出为 SVG 矢量图
- ✅ **复制 Base64** - 一键复制 Base64 数据
- ✅ **历史记录** - 自动保存最近生成的二维码（最多 20 条）
- ✅ **响应式设计** - 适配桌面端和移动端

## 使用方法

### 1. 基本使用

1. 在输入框中输入文本、URL、电话号码等内容
2. 二维码会自动生成并显示在预览区
3. 使用右侧控制面板调整样式

### 2. 自定义样式

- **纠错等级**: 选择纠错等级（越高越容错，但容量越小）
- **尺寸**: 拖动滑块调整二维码大小
- **前景色/背景色**: 使用颜色选择器或直接输入十六进制颜色值
- **预设配色**: 点击预设配色方案快速切换
- **中心 Logo**: 点击上传区域上传图片作为 Logo

### 3. 高级选项

- **点样式**: 方形、圆角、菱形
- **角样式**: 方形、圆角、圆形
- **渐变色**: 启用后设置渐变起始色和结束色

### 4. 导出

- **下载 PNG**: 点击下载 PNG 格式图片
- **下载 SVG**: 点击下载 SVG 矢量图
- **复制 Base64**: 复制图片的 Base64 数据到剪贴板

### 5. 历史记录

- 每次生成的二维码会自动保存到历史记录
- 点击历史记录中的缩略图可快速恢复
- 鼠标悬停可显示删除按钮

## API 参考

### QRCode 类

```javascript
var qr = new QRCode()
qr.generate('Hello World', QRCode.prototype.ECC_M) // 生成二维码
var matrix = qr.getMatrix()  // 获取矩阵数据
var size = qr.getSize()      // 获取矩阵大小
var version = qr.getVersion() // 获取版本号

// 渲染到 Canvas
var canvas = qr.toCanvas(document.getElementById('myCanvas'), {
  width: 280,
  fgColor: '#000000',
  bgColor: '#ffffff'
})

// 获取 DataURL
var dataURL = qr.toDataURL({ width: 280 })

// 获取 SVG
var svg = qr.toSVG({ width: 280 })

// 获取 Blob
qr.toBlob(function(blob) {
  // 处理 blob
}, { width: 280 })
```

### 纠错等级

| 常量 | 值 | 纠错能力 |
|------|-----|---------|
| `ECC_L` | 1 | 低 (7%) |
| `ECC_M` | 0 | 中 (15%) |
| `ECC_Q` | 3 | 较高 (25%) |
| `ECC_H` | 2 | 高 (30%) |

## 技术实现

- 纯 JavaScript 实现 QR 码编码（无第三方依赖）
- 支持版本 1-10 的 QR 码生成
- 完整的 Reed-Solomon 纠错编码
- 8 种掩模模式自动选择最优
- Canvas 2D API 渲染
- SVG 矢量图生成
- localStorage 存储历史记录
