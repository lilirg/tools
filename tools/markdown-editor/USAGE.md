# 📝 Markdown 预览 & 编辑工具

> 纯前端 Markdown 编辑器，支持实时预览、语法高亮、导出 HTML

## 功能特性

- **实时预览**：输入即预览，支持自动/手动刷新
- **工具栏**：快速插入 Markdown 语法（加粗、斜体、标题、链接、图片、代码、列表、引用、表格）
- **常用模板**：空白文档、README 模板、笔记模板、待办事项
- **主题切换**：浅色/深色主题
- **字号调节**：支持 12px-20px 字号选择
- **导出 HTML**：将 Markdown 导出为完整 HTML 文件
- **自动保存**：自动保存草稿到本地存储
- **字数统计**：实时显示字数、词数、行数
- **历史记录**：自动保存导出记录

## 使用方法

### 1. 编辑内容

- 在左侧编辑区输入 Markdown 内容
- 使用工具栏按钮快速插入语法
- 支持 Tab 键插入空格

### 2. 预览

- 开启"自动预览"开关，输入即更新预览
- 点击"预览"按钮手动刷新预览

### 3. 使用模板

- 从模板下拉列表选择预设模板
- 选择后自动填充到编辑器

### 4. 导出 HTML

- 点击"导出 HTML"按钮
- 自动下载为 `.html` 文件
- 包含完整的 HTML 结构和样式

### 5. 主题切换

- 在主题下拉列表选择"浅色主题"或"深色主题"
- 自动保存选择偏好

## 支持的 Markdown 语法

| 语法 | 说明 |
|------|------|
| `# 标题` | 1-6 级标题 |
| `**加粗**` | 加粗文本 |
| `*斜体*` | 斜体文本 |
| `~~删除线~~` | 删除线 |
| `- 列表` | 无序列表 |
| `1. 列表` | 有序列表 |
| `- [ ] 任务` | 任务列表 |
| `> 引用` | 引用块 |
| `` `代码` `` | 行内代码 |
| ` ```代码块``` ` | 代码块（支持语言标注） |
| `[链接](url)` | 超链接 |
| `![图片](url)` | 图片 |
| `\| 表格 \|` | 表格 |
| `---` | 水平线 |

## 技术说明

- 纯前端实现，零外部依赖
- 自研 Markdown 解析引擎，支持常用语法
- 使用 `localStorage` 保存草稿和设置
- 导出 HTML 使用内置模板
