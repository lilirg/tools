✍️ 手写签名插件

CanvasEngine + SignaturePad · 原生无依赖 · 支持鼠标/手写笔 · 自动上传

// 方式一：直接绑定已有 canvas 元素
var canvas = document.getElementById('signatureCanvas')
var engine = new CanvasEngine(canvas, {
  penColor: '#1a1a1a',
  velocitySensitive: true
})

// 获取签名数据
engine.getSignatureData() // → base64
engine.getSignatureFile() // → File 对象
engine.clear() // 清空
engine.undo() // 撤销
