# 🎨 颜色选择器 & 调色板 - 使用文档

## 功能概述

纯前端实现的颜色选择器和调色板生成工具，支持多种颜色格式互转、配色方案生成、图片取色和对比度检测。

## 功能列表

### 1. 🎨 颜色选择
- **色相选择**：拖拽色相条选择色相（0°-360°）
- **饱和度/亮度选择**：在二维面板上选择饱和度和亮度
- **颜色预览**：实时显示当前选中的颜色
- **多格式显示**：同时显示 HEX、RGB、HSL、HSV 四种格式
- **颜色值复制**：点击任意颜色值即可复制到剪贴板
- **颜色值输入**：支持手动输入 HEX/RGB/HSL/HSV 颜色值
- **预设颜色**：25 个常用预设颜色快速选择
- **收藏功能**：收藏常用颜色，方便下次使用

### 2. 🌈 调色板生成
- **互补色**：生成主色的互补色
- **单色系**：基于主色生成不同亮度的单色方案
- **三色系**：生成 120° 间隔的三色方案
- **四色系**：生成 90° 间隔的四色方案
- **类似色**：生成色相环上相邻的颜色方案
- **颜色数量**：可调整生成的颜色数量（2-10 个）
- **颜色复制**：点击色块即可复制颜色值

### 3. 📷 图片取色
- **图片上传**：支持 JPG/PNG/WebP 格式
- **自动提取**：自动提取图片中的主色调
- **颜色预览**：显示提取的颜色及其占比
- **快速应用**：点击提取的颜色即可应用到选择器

### 4. 🔍 对比度检测
- **颜色选择**：使用原生颜色选择器选择前景色和背景色
- **对比度计算**：自动计算 WCAG 标准对比度
- **标准检测**：检测是否符合 AA/AAA 标准
- **实时更新**：颜色变化时自动重新计算

## 操作说明

### 选择颜色
1. 拖拽顶部的色相条选择色相
2. 在饱和度/亮度面板上点击或拖拽选择具体颜色
3. 右侧实时显示颜色预览和四种格式的值
4. 点击颜色值可复制到剪贴板

### 使用调色板
1. 切换到"调色板" Tab
2. 选择配色方案类型（互补/单色/三色/四色/类似）
3. 调整颜色数量
4. 点击色块复制颜色值

### 从图片提取颜色
1. 切换到"图片取色" Tab
2. 点击上传区域选择图片
3. 自动提取并显示主色调
4. 点击提取的颜色应用到选择器

### 检测对比度
1. 切换到"对比度" Tab
2. 使用颜色选择器或输入 HEX 值设置前景色和背景色
3. 自动显示对比度比值和 WCAG 标准检测结果

## 技术说明

- **零外部依赖**：所有颜色转换算法纯数学实现
- **Canvas 绘制**：色相条和饱和度/亮度面板使用 Canvas 2D API
- **颜色格式**：支持 HEX、RGB、HSL、HSV 四种格式互转
- **WCAG 标准**：符合 Web Content Accessibility Guidelines 2.1
- **响应式设计**：适配桌面和移动端
- **数据持久化**：收藏和历史记录保存在 localStorage
