# 🌐 IP / 网络信息查询工具

> 纯前端网络工具，支持 IP 查询、CIDR 计算、子网划分、本机网络信息

## 功能特性

### 🔍 IP 查询
- IPv4 / IPv6 地址格式验证
- IP 地址类别识别（A/B/C/D/E 类）
- 公有/私有地址判断
- 二进制、十进制、十六进制转换
- 回环/广播/组播地址检测

### 📐 CIDR 计算
- 子网掩码计算
- 网络地址、广播地址计算
- 可用主机范围计算
- 通配符掩码显示
- 二进制掩码显示

### 🔀 子网划分
- 基于 CIDR 的子网划分
- 显示每个子网的详细信息
- 支持任意前缀长度

### 💻 本机信息
- 本机 IP 地址（通过 WebRTC 获取）
- 网络状态（在线/离线）
- 连接类型（4G/3G/2G/WiFi）
- 下行速度、网络延迟
- 浏览器、操作系统、语言信息

### 📖 参考
- IP 地址类别表
- 私有地址范围表
- CIDR 前缀与子网掩码对照表

## 使用方法

### IP 查询
1. 切换到"IP 查询"标签
2. 输入 IP 地址（支持 IPv4 和 IPv6）
3. 点击"查询"按钮或按回车键
4. 查看详细的 IP 信息

### CIDR 计算
1. 切换到"CIDR 计算"标签
2. 输入 CIDR 地址（例如：192.168.1.0/24）
3. 点击"计算"按钮
4. 查看子网掩码、网络地址、广播地址、主机范围等

### 子网划分
1. 切换到"子网划分"标签
2. 输入网络 CIDR 和新前缀长度
3. 点击"划分子网"按钮
4. 查看所有子网的详细信息

### 本机信息
1. 切换到"本机信息"标签
2. 自动显示本机网络信息
3. 刷新页面可重新获取

## 技术说明

- 纯前端实现，零外部依赖
- IP 验证使用正则表达式和位运算
- 子网计算使用位运算（32 位整数）
- 本机 IP 通过 WebRTC 的 ICE 候选获取
- 网络信息通过 `navigator.connection` API 获取

## 浏览器兼容

- Chrome 50+
- Firefox 50+
- Safari 10+
- Edge 15+
- IE 不支持（WebRTC 限制）
