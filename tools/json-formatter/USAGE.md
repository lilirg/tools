# 🔄 JSON 格式化 & 对比 - 使用文档

## 功能概述

纯前端实现的 JSON 处理工具，支持格式化、压缩、校验、语法高亮、差异对比和路径查询。

## 功能列表

### 1. 🔧 JSON 格式化
- **格式化**：美化 JSON 输出，支持 2/4/8 空格和 Tab 缩进
- **压缩**：去除所有空白和换行的紧凑格式
- **校验**：检测 JSON 语法错误并定位错误位置
- **语法高亮**：区分键、字符串、数字、布尔值、null
- **双视图**：代码视图和树形视图自由切换
- **路径查询**：支持 `$.key.subkey` 和 `$[0].key` 格式

### 2. 🔍 JSON 差异对比
- **并排对比**：左右并排显示差异
- **行内对比**：逐条显示差异详情
- **差异类型**：新增（绿色）、删除（红色）、修改（黄色）
- **路径显示**：显示差异发生的完整路径

### 3. 📋 其他功能
- **复制结果**：一键复制格式化/压缩结果
- **历史记录**：自动保存操作历史
- **快捷键**：Ctrl+Enter 快速格式化

## 操作说明

### 格式化 JSON
1. 在左侧输入区粘贴 JSON 内容
2. 选择缩进格式（2/4/8 空格或 Tab）
3. 选择视图模式（代码视图/树形视图）
4. 点击"格式化"按钮
5. 右侧显示格式化结果

### 压缩 JSON
1. 输入 JSON 内容
2. 点击"压缩"按钮
3. 右侧显示压缩后的紧凑格式

### 校验 JSON
1. 输入 JSON 内容
2. 点击"校验"按钮
3. 语法正确显示成功提示，错误时显示错误信息

### 路径查询
1. 输入 JSON 内容并格式化
2. 在路径查询输入框中输入路径表达式
3. 支持格式：`$.name`、`$.users[0].email`、`$[0].id`
4. 点击"查询"按钮查看结果

### 对比 JSON
1. 切换到"对比" Tab
2. 在左右两侧分别粘贴 JSON 内容
3. 选择对比模式（并排/行内）
4. 点击"对比"按钮
5. 查看差异结果

## 技术说明

- **零外部依赖**：纯原生 JavaScript 实现
- **语法高亮**：基于正则表达式的语法着色
- **树形视图**：使用 HTML5 `<details>` 元素实现可折叠树
- **差异算法**：递归深度比较，支持对象/数组/基本类型
- **路径查询**：简易版 jq 实现，支持点号和括号访问
- **响应式设计**：适配桌面和移动端
- **数据持久化**：历史记录保存在 localStorage
