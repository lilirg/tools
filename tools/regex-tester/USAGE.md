# ⏱️ 正则表达式测试工具 - 使用文档

## 功能概述

纯前端实现的正则表达式测试工具，支持实时匹配、捕获组展示、替换测试和常用模板库。

## 功能列表

### 1. 🔤 正则表达式输入
- **实时匹配**：输入即匹配，结果实时高亮显示
- **语法校验**：自动检测正则语法错误
- **修饰符开关**：支持 g/i/m/s/u/y 六个修饰符
- **正则显示**：实时显示完整的 `/pattern/flags` 格式

### 2. 📄 测试文本
- **文本输入**：支持多行文本输入
- **匹配高亮**：所有匹配项以黄色高亮显示
- **悬停提示**：鼠标悬停在高亮文本上显示匹配详情

### 3. 📊 匹配结果
- **匹配计数**：显示匹配总数
- **匹配列表**：逐条显示每个匹配的详细信息
- **位置信息**：显示每个匹配的起始和结束位置
- **捕获组**：显示每个匹配的捕获组内容

### 4. 🔄 替换测试
- **替换输入**：输入替换文本
- **替换执行**：点击替换按钮执行替换
- **结果预览**：显示替换后的文本
- **替换统计**：显示替换次数

### 5. 📝 常用模板
- **10 个常用模板**：邮箱、手机号、URL、IP、日期等
- **一键填充**：选择模板自动填充正则和修饰符
- **模板描述**：每个模板都有功能说明

### 6. 📖 语法参考
- **5 大分类**：字符匹配、量词、位置匹配、分组与引用、字符类
- **完整表格**：包含所有常用元字符和语法
- **可折叠**：点击展开/收起

## 操作说明

### 基本使用
1. 在"正则表达式"输入框中输入正则模式（不需要包含分隔符 `/`）
2. 选择需要的修饰符（g/i/m/s/u/y）
3. 在"测试文本"区域输入要测试的文本
4. 匹配结果实时显示在高亮区域和匹配列表中

### 使用模板
1. 从"常用正则模板"下拉列表中选择模板
2. 自动填充正则表达式和推荐修饰符
3. 根据需要调整修饰符

### 替换测试
1. 在"替换为"输入框中输入替换文本
2. 点击"替换"按钮
3. 查看替换结果和替换次数

### 查看捕获组
- 匹配列表中会显示每个匹配的捕获组信息
- 捕获组以 `$1`, `$2` 等标签显示

## 技术说明

- **零外部依赖**：使用原生 `RegExp` 对象
- **实时匹配**：使用防抖（200ms）避免频繁计算
- **匹配高亮**：使用 `<mark>` 标签渲染高亮
- **捕获组提取**：自动解析正则中的捕获组
- **语法参考**：内置完整的正则语法速查表
- **响应式设计**：适配桌面和移动端
- **数据持久化**：历史记录和上次输入保存在 localStorage
